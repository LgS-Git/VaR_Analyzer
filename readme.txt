The VaR_Analyzer aids in the analysis and performance of a Value at Risk analysis for Operational Risks.
The project was based on the simplified Workflow of the Risk Management Consultancy Firm 'Fintegral', as presented during a Workshop at Goethe-University Frankfurt.

The main access point is the Jupyter notebook 'VaR_Analyzer.ipynb' in this directory. The notebook, utilizes several python files, which can be reviewed inside the src/ directory.
To import own data or retrieve results, visit the /data directory.
The notebook is annoted and the functionality is explained within the notebook.

Necessary Python packages to run the Jupyter notebook are:
'numpy', 'pandas', 'matplotlib.pyplot', 'scipy', 'ipywidgets', 'ipython.display'
